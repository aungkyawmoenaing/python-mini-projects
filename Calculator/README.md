# Calculator Project

A modular Python calculator demonstrating clean code, structured design, and robust error handling.  
Supports addition, subtraction, multiplication, and division, logs all operations, and handles invalid inputs gracefully.

## Features

- Addition, subtraction, multiplication, and division  
- Modular design with `calculator.py` for core functions and `logger.py` for logging  
- Logs all operations and errors in `logs/app.log`  
- Handles invalid input and division by zero gracefully  

## Usage

Ensure Python 3.13.4 is installed. Navigate to the calculator folder and run:

```bash
python main.py
